# Sistema de Gestión de Usuarios en JavaScript

## Descripción
Aplicación web desarrollada en JavaScript que permite gestionar una lista de usuarios, agregando y eliminando registros de manera dinámica.

## Funcionalidades
- Agregar usuarios
- Mostrar lista de usuarios
- Eliminar usuarios
- Validación de entradas vacías

## Tecnologías utilizadas
- HTML
- JavaScript

## Cómo ejecutar el proyecto
Abrir el archivo `index.html` en cualquier navegador web.
